# openstack_api
Nghiên cứu khoa học!

### Run guide

```
git clone git@github.com:mr2uang/openstack_api.git
cd openstack_api
pip install -r requirements.txt
python2 api.py
http://localhost:5000/ # De login -> show token and tenant_token
http://localhost:5000/flavors
http://localhost:5000/images
```
